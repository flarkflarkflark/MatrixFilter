## MatrixFilter Free v1.0.0 — Free Release

### Short description:
Free release of MatrixFilter — a high-performance C++ audio filter plugin with matrix routing and preset support.

### Full release notes:

#### What’s new
- Initial free release of MatrixFilter: a multi-channel matrix routing filter plugin implemented in optimized C++ with a CMake build system.
- Core features:
  - Multi-channel matrix routing with configurable routing matrix
  - Per-slot filter types (low-pass, high-pass, band-pass, notch) with independent cutoff/Q
  - Modulation inputs and LFO sources per slot
  - Preset manager with factory and user presets
  - Low-latency DSP path and efficient memory usage

#### Improvements
- Optimized DSP path for low CPU usage
- Cross-platform CMake build system and improved build scripts
- Hi-DPI GUI scaling and improved stability

#### Fixes
- Resolved parameter automation glitches during buffer-size changes
- Fixed rare crash when loading malformed preset files

#### Known issues
- Some hosts may show minor GUI scaling quirks on extreme DPI settings
- AU sandboxing on macOS may require user to enable mic/audio permissions for standalone builds

### Build / Install
- Prebuilt binaries are suggested below; attach when available.
- Build from source (CMake):
  1. git clone https://github.com/flarkflarkflark/MatrixFilter.git
  2. mkdir build && cd build
  3. cmake -S .. -B . -DCMAKE_BUILD_TYPE=Release
  4. cmake --build . --config Release
